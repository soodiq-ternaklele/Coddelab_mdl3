# Data Diri App

Aplikasi Java sederhana untuk input dan output data diri pengguna.

##  Deskripsi
Program konsol interaktif yang memungkinkan pengguna memasukkan nama dan usia, kemudian menampilkannya dalam format yang rapi. Dibuat untuk keperluan pembelajaran pemrograman Java dasar.

##  Fitur
-  Input data nama pengguna
- Input data usia pengguna
-  Tampilan output yang terstruktur
-  Antarmuka konsol yang user-friendly

## 🛠️ Teknologi
- **Bahasa**: Java
- **Tools**: Java Scanner untuk input
- **Platform**: Cross-platform (Windows, Mac, Linux)
